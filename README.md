# MY.GAMES Account Checker

## Что это

Чекер для проверки валидности аккаунтов на my.games. Загружаешь файл с аккаунтами в формате `email:password`, запускаешь проверку, получаешь два файла - с валидными и невалидными аккаунтами.

## Как устроен

Скрипт работает через апи my.games 

**Основная логика:**
1. Читает файл с аккаунтами (по одной строке `email:password`)
2. Для каждого аккаунта делает два запроса:
   - `/api/v3/pub/auth/init` - инициализация авторизации
   - `/api/v3/pub/auth/verify` - проверка логина и пароля
3. Если ответ 200 OK - аккаунт валидный, иначе - невалидный
4. Сохраняет результаты в два файла: `filename_valid.txt` и `filename_invalid.txt`

## Структура проекта

```
├── main.py                 # запускает гуи
└── scripts/
    ├── checker.py          # логика проверки аккаунтов
    └── gui.py              # интерфейс 
```

**checker.py** - класс `AccountChecker`:
- `verify_account()` - проверяет один аккаунт
- `check_accounts_list()` - обрабатывает список аккаунтов
- `write_log()`, `log_request()`, `log_response()` - логирование

**gui.py** - класс `MainWindow`:
- GUI на tkinter с выбором файла, настройками задержки, логами
- Запускает проверку в отдельном потоке, чтобы интерфейс не зависал

## Как запустить

Просто запусти `main.py`:
```bash
python main.py
```

Откроется окно, где можно:
- Выбрать файл с аккаунтами
- Настроить задержку между запросами
- Включить подробные логи (для отладки)
- Сохранять логи в файл

После проверки появится два файла рядом с исходным файлом с аккаунтами.
